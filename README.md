# OCAgent

An Open-source AI Agent Framework for building and deploying autonomous agents.

## Features

- Framework for creating autonomous agents with tools
- Support for various LLM providers
- Easy integration with external APIs and tools
- Modular architecture for customization
- Built-in agent memory and reasoning capabilities

## Getting Started

To get started with OCAgent:

1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Configure your `.env` file with API keys
4. Run an example agent using `python examples/example_agent.py`

## Architecture

The core components of OCAgent include:
- Agent class that orchestrates behavior
- Tool system for executing external tasks
- LLM integration for natural language processing
- Memory system for storing agent experiences
